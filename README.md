# 🏋️ Fitness Advisor

Fitness Advisor is a Python-based health and fitness application that I developed as my final project for Code in Place 2026, a programming course offered by Stanford University.

The program helps users track important health metrics, estimate calorie requirements, monitor fitness habits, and receive personalized recommendations based on their goals. Through this project, I applied the Python concepts I learned during the six-week Code in Place journey to create a practical, interactive application.

---

## Features

### 📊 BMI Calculator

Calculates Body Mass Index (BMI) using a user's height and weight and classifies the result into standard health categories:

* Underweight
* Normal Weight
* Overweight
* Obese

### 🔥 BMR Calculator

Calculates Basal Metabolic Rate (BMR), which estimates the number of calories the body needs at rest.

### 🍽️ Daily Calorie Requirement Estimator

Uses BMR and activity level to estimate daily calorie needs for maintaining weight.

### 🚶 Activity Level Selection

Allows users to choose from different activity levels:

* Sedentary
* Lightly Active
* Moderately Active
* Very Active

### 🎯 Fitness Goal Recommendations

Provides personalized suggestions based on fitness goals such as:

* Weight Loss
* Weight Maintenance
* Weight Gain

### 💧 Water Intake Tracking

Tracks daily water consumption and encourages healthy hydration habits.

### 🏃 Exercise Tracking

Allows users to record exercise activity as part of their fitness routine.

### 📈 Weight History Tracking

Stores and analyzes weight history entered by the user.

### 🔮 Weight Prediction

Predicts future weight trends using previous weight records.

### 💡 Personalized Fitness Advice

Generates customized recommendations based on the user's health data and fitness goals.

## Installation

### Prerequisites

* Python 3.x

### Clone the Repository

```bash
git clone https://github.com/kartikmishra543/fitness-advisor.git
cd fitness-advisor
```

---

## Running the Program

Run the program using:

```bash
python main.py
```

or

```bash
python3 main.py
```

Follow the prompts displayed in the terminal and enter the required information when requested.

---

## Sample Workflow

1. Enter personal information
2. Select activity level
3. Choose a fitness goal
4. View BMI and BMI category
5. View BMR calculation
6. Check estimated calorie requirements
7. Track water intake and exercise activity
8. Enter weight history
9. View predicted future weight
10. Receive personalized fitness recommendations
