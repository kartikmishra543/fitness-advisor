# Fitness Advisor

A Python-based health and fitness tracker developed as the final project for Stanford University's Code in Place 2026.

Fitness Advisor helps users understand their health metrics by calculating BMI (Body Mass Index), BMR (Basal Metabolic Rate), daily calorie requirements, and providing personalized fitness recommendations based on individual goals.

## Features

✅ BMI Calculator

- Calculates Body Mass Index using user height and weight
- Categorizes BMI status (Underweight, Normal, Overweight, etc.)

✅ BMR Calculator

- Calculates Basal Metabolic Rate
- Supports personalized calorie estimation

✅ Daily Calorie Requirement Estimator

- Estimates maintenance calories
- Helps users understand energy needs

✅ Personalized Fitness Recommendations

- Weight loss guidance
- Weight maintenance guidance
- Muscle gain recommendations
