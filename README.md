## Overview

Fitness Advisor is a Python application that I created as my final project for Code in Place 2026.

I wanted to build something practical that combines programming with health and fitness. The program helps users track information such as calories, steps, water intake, exercise habits, and weight history while also calculating BMI and BMR.

Building this project gave me the opportunity to apply many of the Python concepts I learned during Code in Place and see how programming can be used to solve everyday problems.
